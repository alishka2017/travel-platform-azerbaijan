# 📋 Инструкция для владельца сайта (Non-Technical Guide)

Этот гид написан простым языком. Вам не нужно быть программистом, чтобы управлять сайтом.

Все данные сайта хранятся в папке `src/lib/`. Это обычные текстовые файлы JSON (похожие на Excel, но в тексте).

---

## 1. Как изменить текст (туры, рестораны, контакты)

### Шаг 1: Найдите нужный файл
Все файлы находятся в папке `src/lib/`:

| Что менять | Файл |
| :--- | :--- |
| **Туры** | `src/lib/tours.json` |
| **Рестораны** | `src/lib/restaurants.json` |
| **Контакты (WhatsApp, телефон)** | `src/lib/settings.json` |

### Шаг 2: Откройте файл
Используйте любой текстовый редактор (Блокнот, VS Code, Notepad++).

### Шаг 3: Найдите нужный элемент
Пример для тура (файл `tours.json`):
```json
{
  "tours": [
    {
      "id": "baku-city-tour",
      "name": {
        "en": "Baku City Tour",
        "ru": "Городской тур по Баку"
      },
      "description": {
        "en": "Explore the historic Old City...",
        "ru": "Исследуйте исторический Старый город..."
      }
    }
  ]
}
```

### Шаг 4: Измените текст
- Для **английской версии**: измените текст в кавычках после `"en"`.
- Для **русской версии**: измените текст в кавычках после `"ru"`.
- Сохраните файл.

## 2. Как заменить картинку

### Вариант А: Использовать готовую картинку из интернета (просто)
1. Найдите картинку в Google Images или Unsplash.
2. Скопируйте **прямую ссылку** (должна заканчиваться на .jpg, .png).
3. В файле `tours.json` или `restaurants.json` найдите поле `"image"`.
4. Вставьте ссылку в кавычках:
   ```json
   "image": "https://images.unsplash.com/photo-123..."
   ```

### Вариант Б: Загрузить свою картинку (лучше для SEO)
1. Скачайте картинку на компьютер.
2. Переименуйте её в понятное имя (например, `tour-baku-main.jpg`).
3. Положите картинку в папку `public/images/` (создайте папки, если их нет).
4. В файле JSON укажите путь:
   ```json
   "image": "/images/tour-baku-main.jpg"
   ```

## 3. Как добавить новый тур

1. Откройте `src/lib/tours.json`.
2. Скопируйте одну строчку (все от `{` до `}` внутри массива `tours`).
3. Вставьте её после последней строчкой, поставив запятую.
4. Измените `id` (уникальный код, без пробелов), `name`, `description`.
5. Обязательно поставьте запятую между элементами массива (кроме последнего).

**Пример добавления:**
```json
{
  "tours": [
    { ...старый тур... },
    { ...другой тур... },
    { <-- вставьте сюда
      "id": "new-tour-id",
      "name": { "en": "New Tour", "ru": "Новый Тур" },
      ...
    }
  ]
}
```

---

## 4. Как удалить ресторан

1. Откройте `src/lib/restaurants.json`.
2. Найдите блок ресторана, который хотите удалить.
3. Удалите этот блок целиком (от `{` до `}`).
4. **Важно:** Если удаляемый блок был не последним, убедитесь, что у предыдущего блока есть запятая в конце, а у следующего — нет.

---

## 5. Как поменять WhatsApp

1. Откройте `src/lib/settings.json`.
2. Найдите строку `"whatsapp": "+994 ..."`
3. Замените номер на свой.
4. Сохраните файл.

---

## 6. Как сделать Deploy (Опубликовать изменения)

После любого изменения файла нужно "запушить" (отправить) изменения на сервер.

1. **Если вы работаете через GitHub (рекомендуется):**
   - Закоммитьте изменения (сохраните в историю).
   - Нажмите "Push" (отправить) в вашей программе (GitHub Desktop или терминал).

2. **Автоматический деплой:**
   Как только изменения попадут в GitHub, Netlify (хостинг) автоматически обновит сайт в течение 1-2 минут.

---

## 7. Как откатить изменения (Если что-то сломалось)

Если сайт сломался после вашего обновления:

1. Зайдите в GitHub (веб-сайт).
2. Найдите коммит (сохранение) до того, как всё сломалось.
3. Нажмите "Rollback" или "Restore".
4. Сайт вернется к рабочей версии.

---

## 8. Структура данных (для справки)

Все настройки в одном месте:
- `src/lib/tours.json` — список туров.
- `src/lib/restaurants.json` — список ресторанов.
- `src/lib/settings.json` — контакты и настройки сайта.

**Важно:** Не меняйте структуру полей (`id`, `name`, `en`, `ru`), только значения.
