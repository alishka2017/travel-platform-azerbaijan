# SMARTHOLIDAY.AZ — ПОЛНАЯ ТРАНСФОРМАЦИЯ САЙТА

## Контекст
**Сайт:** https://azerbaijan-travel-platform.vercel.app
**Цель:** Сделать лучший туристический сайт по Азербайджану, готовый к ТОП Google без рекламы.
**Текущий статус:** Брендинг изменен на Smartholiday.az, добавлены контакты, улучшен поиск, создана система контента, но требуется полная трансформация.

## Агентная система (параллельное выполнение)

### Advanced SEO Agent
- Анализ ключевых слов для Азербайджана
- Оптимизация meta-тегов для всех страниц (title, description, OG)
- Schema.org разметка для туров, ресторанов, достопримечательностей
- Внутренняя перелинковка между городами, турами, статьями

### Technical SEO Agent
- Оптимизация sitemap.xml (добавить приоритеты, частоту обновления)
- robots.txt тонкая настройка
- Скорость загрузки (Lighthouse оптимизация)
- Устранение ошибок консоли браузера

### Content Intelligence Agent
- Наполнение городов (Guba, Ganja, Lankaran, Qusar) с глубоким контентом
- Создание 7 дополнительных статей блога (1500+ слов каждая)
- Разработка маршрутов (1 день, 3 дня, 7 дней) с почасовой программой
- Проверка всех текстов на "воду" и замену на реальные факты

### Competitor Intelligence Agent
- Анализ TripAdvisor, Booking, Airbnb, azerbaijan.travel, Lonely Planet, Culture Trip
- Выявление лучших практик UX/UI, контент-стратегий, SEO-подходов
- Рекомендации по внедрению (карточки, фильтры, структура)

### UX/UI Analyst
- Аудит текущего интерфейса (мобильный/десктоп)
- Рекомендации по улучшению карточек (как у TripAdvisor)
- Улучшение фильтров (как у Booking)
- Чистота дизайна (как у Airbnb)
- Тестирование навигации и CTA-кнопок

### Design System Agent
- Создание единой системы компонентов (цвета, типографика, отступы)
- Унификация карточек для городов, туров, ресторанов, достопримечательностей
- Адаптация под мобильные устройства

### Performance Engineer
- Замена `<img>` на `<Image>` из next/image (WebP, lazy load)
- Оптимизация бандла (анализ bundle size)
- Кэширование статических ресурсов
- Минимизация JavaScript

### Security Engineer
- Базовая защита форм (валидация, экранирование)
- Проверка на уязвимости зависимостей (npm audit)
- Настройка HTTP-заголовков безопасности

### Refactoring / OpenCode Agent
- Рефакторинг кодовой базы (устранение дублирования)
- Улучшение структуры данных (places.ts, cities.ts)
- Типизация TypeScript (строгие интерфейсы)
- Создание утилитарных функций

### QA Agent
- Полное тестирование сайта (все кнопки, ссылки, формы)
- Поиск 404-страниц
- Тестирование мобильной версии
- Проверка поиска и автодополнения
- Валидация контента (нет AI-текста, нет пустых блоков)
