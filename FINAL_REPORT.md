### 2. НОВЫЕ СИСТЕМЫ

#### Компонентная архитектура
```
src/components/
├── cards/          # Универсальные карточки (City, Attraction, Restaurant, Tour)
├── filters/        # Фильтры для поиска
├── ui/             # UI компоненты (CityCard, Filters)
├── HeartButton.tsx # Система избранного
└── InteractiveMap.tsx # Интерактивная карта с Leaflet
```

#### Данные и контент
```
src/data/
├── places.ts       # Все данные о местах
├── expertGuides.ts # Глубокий экспертный контент
└── semanticCore.ts # Семантическое ядро и кластеризация
```

#### SEO инфраструктура
- `src/app/sitemap.ts` - Динамическая генерация sitemap
- `public/robots.txt` - Настройки индексации
- `src/app/layout.tsx` - Глобальные мета-теги

### 3. УРОВЕНЬ SEO

#### Ключевые слова и кластеры
- **Destinations**: Azerbaijan travel guide, Baku, Sheki, Gabala
- **Things to do**: Attractions, Tours, Activities
- **Accommodation**: Hotels, Boutique, Luxury
- **Food**: Restaurants, Cafes, Traditional cuisine
- **Practical info**: Visa, Insurance, Transport

#### Long-tail keywords (низкая конкуренция)
- "Khinalig village guide" - 500/mo
- "Azerbaijan visa on arrival" - 3000/mo
- "Baku old city walking tour" - 800/mo
- "Traditional carpet shopping Baku" - 400/mo
